# Custom-Keycap-Designs

I created 3 custom keycap designs: 
- The [Normal Key Cap](./NormalKeyCap.step) is an 18mm x 18mm keycap with a custom logo on the top face.
- The [Tab Key Cap](./TabKeyCap.step) is a 27mm x 18mm keycap with a custom tab icon on the top face.
- The [Space Bar Key Cap](./SpaceKeyCap.step) is a 112.5mm x 18mm keycap with 10 hollow circles that get taller as they get closer together on the top face.
